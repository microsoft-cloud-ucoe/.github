# Organization-wide .github

This repository provides default community health files used across organization repositories when they do not define their own.

## Includes

- CODE_OF_CONDUCT.md
- CONTRIBUTING.md
- SECURITY.md
- SUPPORT.md
- Issue forms and PR template
- Workflow templates for easy adoption (not auto-applied)
- Org profile at `profile/README.md`

## Notes

- CODEOWNERS must live in each repo; we include a template here.
- Workflow templates here must be copied or referenced; they won’t auto-run across repos.
- Consider org-level branch protection rules and required checks in GitHub settings.

# Contributing Guidelines

Thank you for considering a contribution!

- For major changes, open an issue to discuss first.
- Keep PRs focused; add tests and docs where applicable.
- Follow repository-specific setup and coding standards.

## Pull Requests

- Branch from the default branch.
- Ensure CI passes.
- Include a clear description and rationale.

## Reporting Issues

Provide steps to reproduce, expected vs. actual behavior, environment details, and logs if relevant.
